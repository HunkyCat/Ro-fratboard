# My Guitar Fretboard

Interactive guitar fretboard visualizer for exploring scales, tunings, and CAGED positions.

## Features

- Scale visualization across 12-24 frets
- Standard and alternate tuning presets
- Custom tuning input
- Full-scale and CAGED display modes
- Built-in random riff playback for quick practice ideas
- Responsive layout for desktop and mobile

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- Canvas API
- Web Audio API

## Run Locally

Open `index.html` in a browser.

For a local server, you can also run:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Project Structure

- `index.html` - application shell
- `styles.css` - layout and visual styling
- `app.js` - fretboard rendering, controls, and audio logic
- `assets/` - decorative media assets
